# 📱 بيت الريف - Progressive Web App

## ✨ **تم التحويل إلى PWA!**

التطبيق الآن **قابل للتثبيت** على الجوال والكمبيوتر، ويعمل **بدون إنترنت**، ويوفر **تجربة مثل التطبيقات الأصلية**! 🎉

---

## ⚠️ **ملاحظة مهمة للمطورين**

إذا ظهر لك خطأ Service Worker في بيئة Figma Preview:
```
SecurityError: Failed to register a ServiceWorker
```

**هذا طبيعي تماماً!** ✅ 
- Service Worker معطل تلقائياً في بيئة Figma Preview
- سيعمل بشكل كامل على الدومين الإنتاجي: `app.bietalreef.ae`
- راجع ملف `/PWA_DEPLOYMENT_GUIDE.md` للتفاصيل الكاملة

---

## 🚀 **كيف أثبّت التطبيق؟**

### **على الجوال (Chrome/Edge):**
1. افتح الموقع: **https://app.bietalreef.ae**
2. انتظر 5 ثوانٍ → يظهر **زر التثبيت الذهبي** 🟡
3. اضغط **"ثبّت الآن"**
4. التطبيق يُضاف للشاشة الرئيسية ✅

### **على iPhone (Safari):**
1. افتح Safari
2. اضغط زر **المشاركة** (⬆️)
3. اختر **"Add to Home Screen"**
4. التطبيق يُضاف للشاشة الرئيسية ✅

### **على الكمبيوتر (Chrome/Edge):**
1. افتح الموقع
2. في شريط URL، اضغط أيقونة **التثبيت** (+)
3. اختر **"Install"**
4. التطبيق يفتح في نافذة منفصلة ✅

## 🛠️ **للمطورين:**

### **الملفات الأساسية:**
```
/public/manifest.json          → بيانات التطبيق
/public/service-worker.js      → Offline Support
/hooks/usePWA.ts              → PWA Hook
/components/mobile/InstallPWAPrompt.tsx → مكون التثبيت
/App.tsx                      → دمج المكون
```

### **الخطوات المتبقية:**
1. ✅ إضافة الأيقونات في `/public/icons/`
   - **icon-192x192.png** (إلزامي)
   - **icon-512x512.png** (إلزامي)
2. ✅ تفعيل HTTPS (Cloudflare تلقائي)
3. ✅ اختبار التثبيت (Chrome DevTools)
