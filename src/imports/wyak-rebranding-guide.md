# النص الرسمي الرئيسي للمنصة

## نظرة عامة

**وياك** هو وكيل ذكاء اصطناعي تنفيذي يتصل بأدواتك ويقوم بالعمل نيابةً عنك.
بدلاً من مجرد الإجابة على الأسئلة مثل روبوتات الدردشة التقليدية، يقوم **وياك** باتخاذ إجراء فعلي — يمكنه إرسال الرسائل، تحديث البيانات، تجهيز التقارير، متابعة الحسابات، معالجة الملفات، تشغيل الحوسبة السحابية، استخدام المتصفح والكمبيوتر، وإدارة سير العمل الكامل للنشاط التجاري.

يمكنك التفاعل مع **وياك** من خلال المحادثة. صف ما تحتاجه، وسيطرح عليك أسئلة توضيحية، ثم يربط الأدوات والخدمات ذات الصلة، وينفذ المهمة خطوة بخطوة. كما يمكنه العمل تلقائيًا عبر **المحفزات** أو الجداول الزمنية أو أحداث البريد والرسائل أو إشعارات الويب — حتى أثناء غيابك.

يتكامل **وياك** مع آلاف التطبيقات والخدمات، ويمكن ربطه بأي واجهة برمجة تطبيقات HTTP أو أي خادم MCP. كما يمتلك **كمبيوتره السحابي الخاص**، مما يمكّنه من تشغيل التعليمات البرمجية، معالجة الملفات، تصفح الإنترنت، وتشغيل البرامج أو المنصات التي لا توفر API مباشر.
وبفضل قدرته على التفكير المرحلي وتحويل كل طلب إلى **مهمة واضحة**، يستطيع **وياك** التعامل مع التعقيدات والحالات الاستثنائية التي تعجز عنها أدوات الأتمتة التقليدية.

تم تصميم **وياك** لأصحاب الأعمال، والمؤسسين، والمديرين، ومزودي الخدمات، وكل من يريد أتمتة أعماله اليومية وإدارة نشاطه التجاري باحترافية.
يتفوق **وياك** خصوصًا في:

* تشغيل الحسابات التجارية
* متابعة العملاء
* إعداد التقارير
* إدارة الحضور الرقمي
* تنفيذ التسويق
* مراقبة المنافسين
* تنظيم العمليات اليومية للفريق أو للمزود

# الاتصالات

## ما هي الاتصالات؟

الاتصالات هي الطريقة التي يستخدمها **وياك** للوصول إلى جميع تطبيقاتك وأدواتك.
يمتلك **وياك** تكاملات مباشرة مع عدد ضخم من الخدمات، كما يمكنه الاتصال بأي:

* API
* خادم MCP
* أو خدمة يتم الوصول إليها عبر المتصفح وكمبيوتر وياك

### كيف تعمل الاتصالات؟

1. **أنت تمنح الوصول**
   يفتح وياك عملية تسجيل دخول آمنة.
2. **أنت تتحكم في الأذونات**
   تحدد ماذا يستطيع وياك فعله.
3. **تصبح الأدوات متاحة**
   بعد التفويض، يستخدم وياك هذه الأدوات لإنجاز المهام.

### نصيحة

إذا قال **وياك** إنه لا يستطيع تنفيذ شيء في خدمة متصلة، فقد يحتاج إلى **تفعيل أدوات إضافية** داخل نفس الاتصال.

* يضغط الأزرار
* يملأ النماذج
* يتنقل بين الصفحات
