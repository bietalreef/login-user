
## ✅ صح

```ts

```

👉 والصورة نفسها موجودة هنا:

```text
/public/assets/hero-construction.webp
```

# 🧠 يعني إيه “هتترفع”؟

## يعني:

1. AI يجيب صورة مناسبة
2. يحفظها كملف
3. يحطها داخل:

```text
/public/assets/
```

4. يعدل الكود يستخدمها

# 🎯 الخلاصة

✔️ الصور لازم تكون ملفات حقيقية

✔️ ومستخدمة في الكود بمسار محلي
